# Instagram Clone (StaticSocialMedia)

Este es un proyecto sencillo de clon de Instagram construido con HTML y CSS.

## Características
- Diseño responsivo (Mobile-first).
- Barra de navegación y menú inferior.
- Feed de publicaciones estático.
- Uso de Font Awesome para iconos.

## Tecnologías
- HTML5
- CSS3
- Font Awesome
